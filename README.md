<h1 align="center">Телеграм бот - SearchOtvet</h1>
<h1>Введение в проект</h2>
<p>SearchOtvetBot - Уникальный бот, который позволяет искать ответы по всему интернету по фото или тексту! <br><br>
В проекте есть: <br>
  <ul dir="auto">
<li>Умеет искать ответы в интернете по запросу, который вы пишете (библиотека BeautifulSoup4)</li>
<li>Умеет читать текст с фотографии и искать по данному тексту ответы в интернете.</li>
<li> Выдает первые 5 самых главных результатов поиска.</li>
</ul>
</p>
